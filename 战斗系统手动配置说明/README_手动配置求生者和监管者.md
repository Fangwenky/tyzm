# 战斗卡牌游戏 - 手动配置求生者和监管者版本 V10.0

## 功能介绍

在这个版本中，您可以在代码中手动配置：
1. **可选求生者列表** - 控制玩家在求生者选择界面中能够选择的角色
2. **可选监管者列表** - 控制游戏中可能出现的监管者角色

**新版本特性**：
- ✅ 隐藏了调试配置提示弹窗，界面更加简洁
- ✅ 新增三个求生者角色：舞女、杂技演员、昆虫学家
- ✅ 所有角色技能均为确定效果，无随机性
- ✅ 完善的角色平衡和技能设计

## 使用方法

### 1. 配置可选角色

打开 `src/main.ts` 文件，找到 `BATTLE_CONFIG` 配置区域：

```typescript
const BATTLE_CONFIG = {
    // 可选择的求生者ID列表（玩家只能从这些求生者中选择）
    availableSurvivors: [
        'doctor',        // 医生
        'lawyer',        // 律师
        'gardener',      // 园丁
        'mechanic',      // 机械师
        'forward',       // 前锋
        'dancer',        // 舞女
        'acrobat',       // 杂技演员
        'entomologist'   // 昆虫学家
    ],

    // 可选择的监管者列表（游戏会从这些监管者中选择）
    availableHunters: [
        { name: '厂长', hp: 180 },
        { name: '小丑', hp: 190 },
        { name: '蜘蛛', hp: 200 }
    ]
};
```

### 2. 新增角色介绍

#### 舞女 💃
- **类型**: 敏捷型
- **生命值**: 95
- **被动技能 - 华丽转身**: 每当使用行动牌时，下一张攻击牌伤害+2
- **主动技能 - 魅惑之舞**: 消耗1行动点，敌人受到魅惑效果，自己获得6点护盾

#### 杂技演员 🤹
- **类型**: 敏捷型
- **生命值**: 100
- **行动点**: 4
- **被动技能 - 灵巧身手**: 受到攻击时，有25%的几率完全闪避伤害并抽1张牌
- **主动技能 - 空中翻腾**: 消耗2行动点，对敌人造成8点伤害，获得1点额外行动点

#### 昆虫学家 🔬
- **类型**: 智力型
- **生命值**: 88
- **被动技能 - 昆虫之友**: 回合开始时，召唤一只昆虫伙伴，对敌人造成3点持续伤害
- **主动技能 - 毒虫释放**: 消耗1行动点，对敌人造成6点伤害，敌人接下来2回合每回合受到4点毒素伤害

#### 机械师 🔧
- **类型**: 智力型
- **生命值**: 92
- **被动技能 - 机械天赋**: 每当使用卡牌时，有30%几率获得2点护盾
- **主动技能 - 机械改造**: 消耗2行动点，从牌库随机获得一张装备牌到手牌

#### 前锋 ⚔️
- **类型**: 力量型
- **生命值**: 105
- **被动技能 - 冲锋陷阵**: 每当对敌人造成伤害时，下一张攻击牌伤害+1
- **主动技能 - 破釜沉舟**: 消耗1行动点，对敌人造成自己已损失生命值25%的伤害，最少8点

### 3. 可选求生者ID列表

您可以从以下求生者中选择添加到配置：
- `doctor` - 医生 👩‍⚕️
- `lawyer` - 律师 👨‍💼
- `gardener` - 园丁 👨‍🌾
- `mechanic` - 机械师 🔧
- `forward` - 前锋 ⚔️
- `dancer` - 舞女 💃
- `acrobat` - 杂技演员 🤹
- `entomologist` - 昆虫学家 🔬
- `novelist` - 小说家 📝
- `wildman` - 野人 🦍
- `reporter` - 记者 📰
- `composer` - 作曲家 🎵
- `little_girl` - 小女孩 👧

### 4. 可选监管者列表

您可以从以下监管者中选择：
- `厂长` - 经典监管者，平衡的血量和技能
- `小丑` - 诡异监管者，拥有特殊能力
- `蜘蛛` - 高血量监管者，擅长持久战
- `红夫人` - 优雅监管者，技能华丽
- `愚人金` - 高防御监管者，血量最高
- `噩梦` - 低血量但有强力回复能力

### 5. 游戏流程

1. 启动游戏后，直接进入求生者选择界面
2. 玩家只能从配置的求生者列表中选择角色
3. 游戏会从配置的监管者列表中随机选择一个作为对手
4. 其他游戏流程保持不变：选择求生者 → 构建牌组 → 开始战斗

### 6. 修改配置后的使用

修改 `main.ts` 中的配置后，需要重新编译：

```bash
npx tsc
```

然后刷新浏览器页面即可看到新的配置生效。

## 技术说明

- 求生者配置通过过滤 `SurvivorSelectionUI` 显示的角色列表来实现
- 监管者配置替换了原有的随机选择逻辑
- 配置验证确保只使用有效的角色ID和监管者名称
- 所有角色技能都经过TypeScript类型检查，确保代码稳定性
