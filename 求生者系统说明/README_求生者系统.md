- **律师** 👨‍💼 - 拥有钢铁意志和策略思维
- **小说家** 📝 - 用文字的力量改变现实
- **记者** 📰 - 善于抓住关键时机
- **作曲家** 🎵 - 用旋律影响战场
- **昆虫学者** 🔬 - 拥有惊人的生存适应能力

#### 敏捷型求生者 🏃
- **舞女** 💃 - 优雅敏捷的表演艺术家
- **杂技演员** 🤹 - 灵活的爆破专家
- **小女孩** 👧 - 神秘的力量守护者

#### 力量型求生者 💪
- **园丁** 👨‍🌾 - 充满父爱的防御专家
- **野人** 🦍 - 原始的力量战士

### 🎯 核心功能特点

#### 1. **精美的求生者选择界面**
- 卡片式设计，每个求生者都有专属头像和背景介绍
- 清晰展示被动技能和主动技能描述
- 响应式设计，支持移动设备
- 丰富的动画效果和交互反馈

#### 2. **独特的技能系统**
每个求生者都拥有：
- **被动技能**: 自动触发的特殊能力
- **主动技能**: 消耗行动点的强力技能
- **属性加成**: 不同的生命值和行动点配置

#### 3. **深度战斗机制**
- 求生者技能与卡牌系统完美融合
- 多样化的技能触发时机（回合开始、造成伤害、受到伤害等）
- 策略性的技能冷却和行动点管理

#### 4. **流畅的游戏流程**
新的游戏流程：**求生者选择** → **牌组构建** → **战斗开始**

## 🎮 游戏玩法说明

### 开始游戏
1. 选择一个求生者角色
2. 构建你的7张卡牌组合
3. 进入战斗，面对随机敌人

### 战斗中的求生者技能
- **被动技能**: 自动触发，无需操作
- **主动技能**: 点击角色面板中的技能按钮使用
- **技能冷却**: 部分技能有冷却时间限制

### 求生者特色示例

#### 医生的救赎之道 👩‍⚕️
- **被动**: 对敌人造成伤害时恢复生命
- **主动**: 强力治疗技能

#### 野人的狂暴之力 🦍
- **被动**: 生命越低伤害越高
- **主动**: 强力攻击，但会影响下回合

#### 小女孩的神秘守护 👧
- **被动**: 将伤害转化为护盾
- **主动**: 基于敌人生命值的百分比伤害
